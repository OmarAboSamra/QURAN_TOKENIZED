## Color Scheme

**Header:**
- Background: `bg-gradient-to-l from-green-600 to-green-700`
- Text: White
- Accent: `bg-green-800 bg-opacity-40` (info badge)

**Info Bar:**
- Background: `bg-gradient-to-l from-green-50 to-white`
- Border: `border-r-4 border-green-600`
- Stats: `text-green-600`

**Verse Cards:**
- Background: `linear-gradient(to left, #f0fdf4, #ffffff)`
- Border: `border-right: 4px solid #10b981`
- Ayah badge: `bg-green-600 text-white`

**Token Cards:**
- Background: `bg-white`
- Border: `border-gray-200`
- Hover: `hover:shadow-md`

**Root Badges:**
- With root: `bg-green-100 text-green-800`
- Without root: `bg-gray-100 text-gray-500`
- Hover: `hover:bg-green-200`

**Pagination:**
- Enabled: `bg-green-600 text-white hover:bg-green-700`
- Disabled: `bg-gray-200 text-gray-400 cursor-not-allowed`

**Search:**
- Focus: `focus:ring-2 focus:ring-green-500`

**Modal:**
- Backdrop: `rgba(0, 0, 0, 0.5) + backdrop-blur(4px)`
- Header: Same as main header gradient
- Highlighted words: `bg-yellow-200`

---

## Typography

**Arabic Text:**
- Font: Amiri, Scheherazade New
- Size: `1.5rem` (24px)
- Line Height: `2.5rem` (40px)
- Weight: 400

**Headings:**
- H1: `text-3xl font-bold` (Amiri)
- H2: `text-2xl font-bold`

**Body:**
- Labels: `text-sm text-gray-600`
- Stats: `text-2xl font-bold text-green-600`
- Badges: `text-sm font-semibold`

---

## Responsive Breakpoints

**Mobile (< 768px):**
- Header: 1-column stack
- Token grid: 1 column
- Pagination: Smaller text

**Tablet (768px - 1024px):**
- Header: 3-column grid
- Token grid: 2 columns

**Desktop (> 1024px):**
- Header: 3-column grid
- Token grid: 3 columns
- Maximum container width: 1280px

## Animations

**Loading Spinner:**
```css
@keyframes spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}
```

**Root Badge Hover:**
```css
.root-badge:hover {
    transform: scale(1.05);
    box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1);
}
```

**Ayah Highlight:**
```javascript
element.style.backgroundColor = '#fef3c7';  // Yellow flash
setTimeout(() => {
    element.style.backgroundColor = '';      // Fade out
}, 1000);
```

**Page Transition:**
```javascript
window.scrollTo({ top: 0, behavior: 'smooth' });
```
